## Enduring Strategic Goals

The long-horizon shape of the project is still:
- a supervised automation runtime
- reliable state/orientation under real UI conditions
- deterministic execution where earned
- recovery and escalation with preserved context
- richer recording for diagnosis and replay
- incremental buildability rather than all-or-nothing architecture

## What This File Should Contain

This file should only preserve:
- durable long-horizon goals
- strategic sequencing that may still matter later
- historical context that explains why certain directions were explored

This file should not be used for:
- current implementation claims
- current script inventory
- current phase status
- asserting that deleted or untrusted code still exists

## Practical Rule

If a statement is about what exists now, it belongs somewhere else.
If a statement is about what the project is ultimately trying to become, it can stay here.
